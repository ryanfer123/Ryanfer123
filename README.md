## How to Use

To run any of the projects in this repository:

1. **Clone the repository**
   ```bash
   git clone https://github.com/ryanfer123/Ryanfer123.git
   cd Ryanfer123
   ```

2. **Open the project folder**
   Each linked project includes setup instructions in its own repository README.

3. **Install dependencies**
   Follow the instructions in the project-specific README or install from `requirements.txt` / `package.json` where applicable.
